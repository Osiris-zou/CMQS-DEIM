# Code Availability Statement

The implementation of CMQS is available at: https://github.com/Osiris-zou/CCAQS_main.

The repository provides the modified DEIM components, configuration files, training and evaluation commands, and scripts for reproducing the main quantitative analyses and visualizations. CMQS is implemented on top of the official DEIM framework, and the original DEIM training pipeline, Dense O2O matching, Matchability-Aware Loss and inference post-processing are preserved. The repository also documents the modified files and the correspondence between the paper notation \(T_{exit}\) and the implementation parameter `query_select_gt_stop_epoch`.
