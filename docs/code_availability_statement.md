# Code Availability Statement

The CMQS-DEIM source code and reproducibility package are publicly available at:

https://github.com/Osiris-zou/CMQS-DEIM

The repository provides the three modified DEIM runtime components required for curriculum epoch propagation and matching-aware query selection, DEIM-S and DEIM-L configurations, dedicated ablation configurations, training and evaluation scripts, environment specifications, profiling utilities, query-analysis tools, plotting scripts, and machine-readable result tables.

The GitHub `v1.0.0` release is prepared to distribute the two principal CMQS model checkpoints and their verified training records:

- DEIM-S + CMQS: 49.27 AP;
- DEIM-L + CMQS: 54.58 AP.

The release links and required asset filenames are documented in `release_assets/README.md`. The COCO dataset and upstream DEIM initialization checkpoints are not redistributed and remain subject to their original licenses. The GitHub v1.0.0 release has been archived on Zenodo under DOI 10.5281/zenodo.20815315. The concept DOI 10.5281/zenodo.20815314 resolves to the latest archived version of the software.
