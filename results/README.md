# Reported Results

The CSV files contain the numeric values reported in the manuscript. They are provided for auditability, plotting and cross-checking against the released model checkpoints and logs.

- `table1_accuracy.csv`
- `table2_recall.csv`
- `table4_ablation.csv`
- `table5_beta_sensitivity.csv`
- `table6_exit_sensitivity.csv`
- `table7_profile.csv`
- `table8_query_analysis.csv`
- `table9_seed_stability.csv`
- `table9_seed_summary.csv`

The main DEIM-S and DEIM-L CMQS checkpoints and verified run records are distributed through the `v1.0.0` GitHub Release. See `../docs/model_zoo.md`.
