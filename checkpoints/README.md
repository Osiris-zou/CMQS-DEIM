# Checkpoints

The two CMQS checkpoints used for the principal DEIM-S and DEIM-L results are distributed as GitHub Release assets rather than committed to Git history:

- [DEIM-S + CMQS, 49.27 AP](https://github.com/Osiris-zou/CMQS-DEIM/releases/download/v1.0.0/cmqs_deim_s_best.pth)
- [DEIM-L + CMQS, 54.58 AP](https://github.com/Osiris-zou/CMQS-DEIM/releases/download/v1.0.0/cmqs_deim_l_best.pth)

The URLs become active after the `v1.0.0` release assets are uploaded. Upstream DEIM initialization checkpoints are not redistributed and must be obtained from the upstream project.
