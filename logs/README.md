# Training Logs

Verified training logs for the two principal CMQS models are distributed as GitHub Release assets:

- [DEIM-S + CMQS log](https://github.com/Osiris-zou/CMQS-DEIM/releases/download/v1.0.0/cmqs_deim_s_logs.txt)
- [DEIM-L + CMQS log](https://github.com/Osiris-zou/CMQS-DEIM/releases/download/v1.0.0/cmqs_deim_l_logs.txt)

Each text file contains the preserved training and validation records for the corresponding CMQS run, including the COCO evaluation metrics and the best-recorded validation result reported in the manuscript.

Private paths, credentials, server addresses, and unrelated terminal output should be removed before release.
