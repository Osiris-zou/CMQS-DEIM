# Training Logs

Verified training records for the two principal CMQS models are distributed as GitHub Release assets:

- [DEIM-S + CMQS logs](https://github.com/Osiris-zou/CMQS-DEIM/releases/download/v1.0.0/cmqs_deim_s_logs.zip)
- [DEIM-L + CMQS logs](https://github.com/Osiris-zou/CMQS-DEIM/releases/download/v1.0.0/cmqs_deim_l_logs.zip)

Each archive should contain the JSON-lines training log, captured console output, configuration snapshot, and terminal COCO evaluation summary. Remove private paths or credentials before release.
